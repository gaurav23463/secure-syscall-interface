Secure System Call Interface

Modern, role-aware system call simulator designed for secure teaching, auditing, and safe file operations.

Login Register

Why this project matters

This interface demonstrates how system calls can be protected by authentication, role-based access control, and secure request validation. It is built to be user-friendly for learners and secure enough to illustrate production-minded design.

Authentication

Users log in with credentials and receive a secure JWT token for every request.

Role-Based Access

Admins view audit logs while regular users are limited to safe file operations.

Audit Logging

Every simulated syscall is recorded in MongoDB with timestamp, action, and outcome.
